# sp-fx-es-next

## Summary

Demonstration of errors causes by using target="esnext" and lib "es6" in tsconfig.json when using Optional Chaining.

Causes error on line 22 of HelloWorldWebPart.ts using Optional Chaining.

Module parse failed: Unexpected token (10:44)
You may need an additional loader to handle the result of these loaders.
